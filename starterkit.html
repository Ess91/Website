The Starter Kit

24th May 2020
I feel like it's been a while since I posted my first blog post! Thank you to everyone that has read my blog post and also those supporting my coding journey.

I wanted my next blog post to be about what I've learnt so far since starting this journey but now this post will be completely different. My really good friend hit me up sometime last week or the week before asking for some advice on starting her coding journey. First I thought what could I possibly share? But I quickly changed that mindset (I'm learning to change that mindset but that's for another blog post lol!).

Anyways, I compiled a list of sites that I thought would be useful and I thought it would be nice to share if anyone is thinking about starting coding. I hope this information helps.

Resources:
Free Code Camp
FREE

This one is pretty good, they have languages for you to learn, Ruby, HTML, CSS etc. There is a lot of material and mini activities/challenges to get through. They also have mini projects at the end of each language helping you to put everything into practice.

W3 Schools
FREE

Lots of information you can get on CSS, HTML and Javascript. However, it is a little outdated so you can use it to a certain extent but it is still very helpful.

Codecademy
FREE

This site is free to an extent, a lot of activities to get through, it is not as detailed as some of the websites I've encountered but I guess it depends on yourself and how you learn. It also has premium features such as portfolio-building which is the Pro Membership but this comes at a cost.

MDN Web Docs
FREE

This is a developer's resource guides developed by developers, where all the information is up to date and the best place to go when needing help.

Udemy
FREE

This site is not free, however they do a lot of flash sales, which you can keep an eye out for. I've bought a few courses at a reduced price; the courses are very clear with videos to help you as well as quizzes and little projects to do at the end.

Code Editors - Beginner Friendly!

CodePen is an online code editor. This is good because it allows you to practice the basics of HTML, CSS and Javascript without the standard structure required. It is also a good way of building a portfolio for yourself just to see how you are doing.

Visual Studio Code - It is very beginner friendly, it sets up the HTML structure for you (which I only learned how to do recently lol). It is also good at seeing errors and where you have placed it and what it needs to be changed.

I use both the CodePen to see my journey and a way for me to build my portfolio and I use the Visual Studio Code to keep practicing and also get into the habit of writing codes in a professional manner.

These are some of the basic resources you could use to help you get started. I will continually add more resources as time goes on.
